

# Embedded Systems Internship - PoulGilan Institute, Guilan

This repository showcases the work completed during my **Embedded Systems Internship** at **PoulGilan Institute** from **July 2025 to September 2025**. During this internship, I gained practical experience in designing, debugging, and developing embedded systems using Arduino and Raspberry Pi platforms.

## Internship Overview

As part of my internship, I contributed to several projects that involved real-time embedded applications. My responsibilities included designing and debugging embedded systems, integrating sensors, and developing communication protocols for data transfer. I also supported the development of a modular sensor integration system, focusing on wireless communication and data management.

### Key Learning Outcomes:

* **Arduino and Raspberry Pi**: Gained hands-on experience in working with Arduino and Raspberry Pi platforms to create real-time embedded systems.
* **PCB Design**: Assisted in designing PCBs for various embedded applications.
* **Microcontroller Programming**: Developed firmware to interface with sensors and actuators.
* **Wireless Data Transfer**: Worked on integrating wireless data transfer for IoT and sensor-based systems.

## Projects:

1. **Real-time Embedded Applications**:

   * Designed and debugged embedded systems using Arduino and Raspberry Pi.
   * Worked on real-time processing tasks, ensuring efficient use of system resources.

2. **Modular Sensor Integration System**:

   * Supported the development of a modular sensor integration system that featured wireless data transfer.
   * Implemented sensor fusion techniques to gather and analyze sensor data from multiple sources.

3. **PCB Design and Microcontroller Programming**:

   * Assisted in designing PCBs to interface various sensors and communication modules.
   * Programmed microcontrollers to handle sensor inputs and control outputs in real-time.



## Technologies Used:

* **Programming Languages**: C, Python
* **Tools**: Arduino IDE, Raspberry Pi OS, KiCad (for PCB design)
* **Platforms**: Arduino, Raspberry Pi
* **Wireless Protocols**: Wi-Fi, Bluetooth, Zigbee (for data transfer)
